Thin Cultural Heritage Artifacts

Geometry-Constrained Bidirectional Registration

A point cloud registration pipeline for thin cultural heritage artifacts, designed to improve alignment stability through geometry-aware constraints and bidirectional correspondence reasoning.

View Visual Results Source Code
Problem

Thin artifact registration

Thin, sheet-like cultural heritage objects create ambiguous geometry and fragile overlap, making point cloud alignment sensitive to drift and incorrect matches.

Method

Bidirectional constraints

The registration process uses forward and reverse matching cues with geometry-aware restrictions to encourage consistent alignment across delicate surfaces.

Output

Visual reconstruction evidence

The rotating result videos show merged point cloud behavior across multiple artifacts, making surface continuity and registration quality easy to inspect.
