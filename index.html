Abstract

Thin cultural heritage artifacts present a difficult point cloud registration problem because their sheet-like geometry often provides limited thickness, ambiguous local correspondences, and fragile surface boundaries. This project presents a geometry-constrained bidirectional registration pipeline that uses structural constraints and bidirectional correspondence reasoning to improve alignment consistency. The visualizations below show merged rotating point clouds for multiple artifact cases.
